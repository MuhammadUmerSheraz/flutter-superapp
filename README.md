# flutter-superapp

Production-ready Flutter Cross-Platform App built with Clean Architecture and SOLID principles.

## Architecture

- **Clean Architecture** - Separation of concerns across domain, data, and presentation layers
- **Modular Structure** - Independently deployable and testable modules
- **SOLID Principles** - Maintainable, extensible codebase

## Tech Stack

Flutter, Dart, Bloc, GetIt, Injectable

## Setup

```bash
flutter pub get
cp .env.example .env
flutter run
```

## Testing

```bash
flutter test
```

## Docker

```bash
docker-compose up -d
```
